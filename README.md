# Thendrask Launcher — Marketing Site

The landing page for [Thendrask Launcher](https://github.com/Sxarlos/ThendraskLauncher), an open-source Minecraft launcher with a dedicated GregTech Hub plus browsing for Modrinth, CurseForge, FTB, ATLauncher, and Technic in one place.

## Stack

- Plain HTML, CSS, and vanilla JS — no build step required
- Hosted via Cloudflare Pages
- Assets: `assets/logo.png`, `assets/og.png`, `assets/screenshots/`

## Deployment

Every push to `main` triggers an automatic deploy via Cloudflare Pages.

To run locally just open `index.html` in a browser — no server needed.
