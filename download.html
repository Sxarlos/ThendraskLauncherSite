Installation steps

STEP 01

Download the installer

Click the button above to download the latest .exe installer from GitHub Releases. If your browser warns you, click "Keep" — it's a code-signed Electron app, not malware.

STEP 02

Run the installer

Double-click the downloaded file and follow the prompts. Choose your install location — a standard Windows installer. Ender Client will launch automatically when done.

STEP 03

Sign in with Microsoft

Log in with your Microsoft account. Your password is entered on Microsoft's own page — Ender Client never sees it. Only an encrypted token is stored locally.

STEP 04

Browse, install, play

Open the modpack browser, pick a source (Modrinth, CurseForge, FTB, ATLauncher, or Technic), install a pack, and hit Play. First launch downloads the game version and assets automatically.

Antivirus false positives

Some antivirus tools flag unsigned or newly published Electron apps. Ender Client is fully open-source — you can read every line on GitHub before running it. If Windows SmartScreen appears, click More info → Run anyway.
