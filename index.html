What's inside

Everything you need to play, nothing you don't.

A focused launcher built with Electron and React. Sign in, find a pack, launch it - with live logs, isolated instances, and the conveniences that usually take three different tools.

Multi-source modpack browser

Search five providers from one window. Sort Modrinth and CurseForge by popular, updated or newest, and filter by category.

Isolated instances

Vanilla or modded, any version, each with its own clean .minecraft folder. No cross-contamination between packs.

Friends & presence

Add friends by code and see who's online and what they're playing - powered by a relay you self-host, so the social graph is yours.

Just an interface

Ender Launcher downloads modpacks and spawns the official Minecraft game. It doesn't modify Minecraft itself, and it isn't affiliated with Mojang or Microsoft. You still need a genuine account to play.

Open source & self-hostable

MIT-licensed and on GitHub. The friends feature runs on a small relay you host yourself - paste its URL into Settings → API Keys and your presence data never leaves infrastructure you control.

Resource usage

Easy on RAM. Hard on features.

Some Electron-based launchers can use significant memory under load. Ender Launcher stays lean - sitting comfortably close to native launchers at idle, well below what CurseForge uses when a modpack launches.

CurseForge up to ~1 GB launching ~200 MB idle
Ender Launcher this launcher ~30–40 MB idle

Measured at idle on Windows. Figures are from real testing - actual usage varies by system, version, and background processes.

Get started

Playing in three steps.

You'll need Java installed to launch the game - Java 21 for 1.20.5+, Java 17 for 1.17–1.20.4, Java 8 for older versions.

STEP 01

Download & install

Grab the latest Windows installer from GitHub Releases and run it. Choose your install location - it's a standard installer.

STEP 02

Sign in & pick a pack

Log in with your Microsoft account, then browse the modpack sources and install whatever you fancy, or create a clean instance.

STEP 03

Hit launch

First launch downloads the version and assets automatically. Watch the progress on the card, then you're in the game.
