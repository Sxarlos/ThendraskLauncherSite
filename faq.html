Frequently asked questions.

Everything you need to know about Ender Client - safety, compatibility, modpack sources, and how it compares to other launchers.

Safety & Trust
Is Ender Client safe?
Yes. Ender Client is fully open-source under the MIT license - every line of code is public on GitHub and can be audited by anyone. It uses Microsoft's official OAuth flow to sign you in, downloads modpacks from trusted sources (Modrinth, CurseForge, FTB, ATLauncher, Technic), and stores only an encrypted refresh token in your OS keychain. Your password is never seen or stored.
Is Ender Client a hacked client or cheat client?
No. "Client" in the name refers to a launcher - like CurseForge or Prism Launcher - not a hacked Minecraft client. Ender Client does not modify Minecraft game files, inject code into the JVM, or provide any in-game advantage. It is simply an interface that downloads modpacks and launches the official game.
Is my Microsoft account safe?
Yes. Ender Client uses Microsoft's official OAuth flow via msmc. Your password is entered on Microsoft's own login page - Ender Client never sees it. Only an encrypted refresh token is stored in your OS keychain, the same way Windows and other apps manage credentials.
Does Ender Client modify my existing Minecraft installation?
No. Each instance gets its own isolated .minecraft folder. Your existing Minecraft installation, the official launcher, and any other third-party launchers you use are completely untouched.
Getting Started
Is Ender Client free?
Yes, completely free. Ender Client is MIT-licensed and open-source with no paid tier, no ads inside the launcher, and no required subscription. If it saves you hassle, you can optionally support development on Buy Me a Coffee or via a Stripe subscription - but that's never required to use the launcher.
How do I install a modpack?
Open Ender Client, sign in with your Microsoft account, then open the modpack browser. Choose a source (Modrinth, CurseForge, FTB, ATLauncher, or Technic), find a pack you like, and hit Install. The launcher handles the rest - downloading the pack, its mods, and the correct Minecraft version. Once complete, click Play from your library.
How do I update Ender Client?
Ender Client checks GitHub Releases automatically. When a new build ships, an update banner appears inside the app within minutes - click it to download and install the update without leaving the launcher.
Compatibility
What modpack sources does Ender Client support?
Modrinth, CurseForge, FTB (modern and legacy), ATLauncher, and Technic - all browsable and installable from a single window inside the launcher. No browser tabs, no manual downloads.
What Java version do I need?
Java requirements are set by Minecraft, not Ender Client:

Java 21 - Minecraft 1.20.5 and newer
Java 17 - Minecraft 1.17 through 1.20.4
Java 8 - Minecraft 1.16.5 and older

You'll need the correct version installed on your system before launching those versions.
Does Ender Client work on Mac or Linux?
Not yet. Ender Client currently targets Windows x64 only. Mac and Linux support is on the roadmap.
Comparisons
Ender Client vs CurseForge - what's the difference?
CurseForge is a large commercial launcher with ads and higher memory usage. Ender Client is ad-free, open-source, and uses far less RAM at idle. It also supports more modpack sources - Modrinth, FTB, ATLauncher, and Technic alongside CurseForge - so you're not limited to one platform. If you're looking for a CurseForge alternative that doesn't serve you ads, Ender Client is built for exactly that.
Ender Client vs Prism Launcher - what's the difference?
Prism is a Qt-based launcher focused on flexibility and very low resource usage - a great choice if you prefer a traditional interface and minimal overhead. Ender Client is Electron/React-based with a more polished, modern interface and built-in social features like friends, presence, and server monitoring. Pick Prism for raw efficiency; pick Ender Client for a curated, visually modern experience with shader auto-install and in-app updates.
Is Ender Client affiliated with Mojang or Microsoft?
No. Ender Client is an independent, community-built project. It is not affiliated with, endorsed by, or associated with Mojang Studios or Microsoft. Minecraft is a trademark of Mojang Studios. You still need a genuine Minecraft account to play.
