Safety & Trust
Is Thendrask Launcher safe?
Yes. Thendrask Launcher is fully open-source under the MIT license - every line of code is public on GitHub and can be audited by anyone. It uses Microsoft's official OAuth flow to sign you in, downloads modpacks from trusted sources (Modrinth, CurseForge, FTB, ATLauncher, Technic), and stores only an encrypted refresh token in your OS keychain. Your password is never seen or stored.
Is Thendrask Launcher a hacked client or cheat client?
No, not at all. Thendrask Launcher does not modify Minecraft game files, inject code into the JVM, or provide any in-game advantage. It is simply an interface that downloads modpacks and launches the official game.
Is my Microsoft account safe?
Yes. Thendrask Launcher uses Microsoft's official OAuth flow via msmc. Your password is entered on Microsoft's own login page - Thendrask Launcher never sees it. Only an encrypted refresh token is stored in your OS keychain, the same way Windows and other apps manage credentials.
Does Thendrask Launcher modify my existing Minecraft installation?
No. Each instance gets its own isolated .minecraft folder. Your existing Minecraft installation, the official launcher, and any other third-party launchers you use are completely untouched.
Getting Started
Is Thendrask Launcher free?
Yes, completely free. Thendrask Launcher is MIT-licensed and open-source with no paid tier, no ads inside the launcher, and no required subscription. If it saves you hassle, you can optionally support development on Buy Me a Coffee or via a Stripe subscription - but that's never required to use the launcher.
How do I install a modpack?
Open Thendrask Launcher, sign in with your Microsoft account, then open the modpack browser. Choose a source (Modrinth, CurseForge, FTB, ATLauncher, or Technic), find a pack you like, and hit Install. The launcher handles the rest - downloading the pack, its mods, and the correct Minecraft version. Once complete, click Play from your library.
How do I update Thendrask Launcher?
Thendrask Launcher checks GitHub Releases automatically. When a new build ships, an update banner appears inside the app within minutes - click it to download and install the update without leaving the launcher.
Compatibility
What modpack sources does Thendrask Launcher support?
Modrinth, CurseForge, FTB (modern and legacy), ATLauncher, and Technic - all browsable and installable from a single window inside the launcher. No browser tabs, no manual downloads.
What Java version do I need?
Java requirements are set by Minecraft, not Thendrask Launcher:

Java 21 - Minecraft 1.20.5 and newer
Java 17 - Minecraft 1.17 through 1.20.4
Java 8 - Minecraft 1.16.5 and older

You'll need the correct version installed on your system before launching those versions.
Does Thendrask Launcher work on Mac or Linux?
Not yet. Thendrask Launcher currently targets Windows x64 only. Mac and Linux support is on the roadmap.
Comparisons
Thendrask Launcher vs CurseForge - what's the difference?
CurseForge is a large commercial launcher with ads and higher memory usage. Thendrask Launcher is ad-free, open-source, and uses far less RAM at idle. It also supports more modpack sources - Modrinth, FTB, ATLauncher, and Technic alongside CurseForge - so you're not limited to one platform. If you're looking for a CurseForge alternative that doesn't serve you ads, Thendrask Launcher is built for exactly that.
Thendrask Launcher vs Prism Launcher - what's the difference?
Prism is a Qt-based launcher focused on flexibility and very low resource usage - a great choice if you prefer a traditional interface and minimal overhead. Thendrask Launcher is Electron/React-based with a more polished, modern interface and built-in social features like friends, presence, and server monitoring. Pick Prism for raw efficiency; pick Thendrask Launcher for a curated, visually modern experience with shader auto-install and in-app updates.
Is Thendrask Launcher affiliated with Mojang or Microsoft?
No. Thendrask Launcher is an independent, community-built project. It is not affiliated with, endorsed by, or associated with Mojang Studios or Microsoft. Minecraft is a trademark of Mojang Studios. You still need a genuine Minecraft account to play.
