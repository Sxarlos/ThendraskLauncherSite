Thendrask Launcher is a free, open-source Minecraft launcher for Windows built to do one thing well - get you into a modpack with as little friction as possible.

Modpack browser

Five sources, one window

Browse Modrinth, CurseForge, FTB (modern and legacy), ATLauncher, and Technic without switching apps or opening a browser. Search, filter by category, and sort by popular, updated, or newest.

  • Sort by popular, updated, or newest
  • Filter by modpack category
  • One-click install from any source

Isolated instances

Every modpack - and every vanilla instance - lives in its own isolated .minecraft folder. Mods from one pack can never bleed into another, and your existing Minecraft install is never touched.

  • Per-instance .minecraft directories
  • Create vanilla instances manually
  • Per-instance play-time tracking

Gameplay & launching

Live launch logs

First run downloads the Minecraft version and assets automatically via minecraft-launcher-core. Progress bars and full game output stream right onto the instance card in real time - no guessing if something is stalled.

Shader auto-install

Thendrask Launcher watches game output for missing EuphoriaPatcher shader errors and automatically pulls the right shaders from Modrinth - no manual hunting through mod pages or version tables.

In-app updates

The launcher polls GitHub Releases on startup. When a new build is available, a banner appears within the app - click once to download and install without opening a browser or losing your place.

Social & server tools

Server monitor

Add servers to watch from inside the launcher. Thendrask Launcher pings each one and reports live status and current player counts - so you know before you launch whether the server is up and if your friends are already on.

  • Live ping and player count
  • Multiple servers at once
  • Status visible without launching the game

Quick connect

A dropdown next to the Play button lists every server saved in your instance's servers.dat. Select one and the game launches straight into it - no menu navigation inside Minecraft.

  • Reads servers.dat - no re-adding required
  • Uses --quickPlayMultiplayer on 1.20+
  • Falls back to --server / --port for older versions

Friends & presence

Add friends by code and see who's online and what they're playing in real time. The social graph runs on a small relay you self-host - paste its URL into Settings and your presence data never leaves infrastructure you control.

  • Add by friend code, no account required
  • See who's online and what they're playing
  • Self-hosted relay - your data stays yours
  • Discord Rich Presence integration

Security & personalisation

Secure Microsoft login

Authentication uses Microsoft's official OAuth flow via msmc. You log in on Microsoft's own page - Thendrask Launcher never sees your password. Only an encrypted refresh token is stored in your OS keychain. Your account is as safe as it is with the official launcher.

  • Official Microsoft OAuth - password never seen
  • Encrypted token in OS keychain
  • Open-source auth code - fully auditable

Seven built-in themes

Swap the entire launcher look in one click. Every theme is built into the app - no downloads, no restarts.
